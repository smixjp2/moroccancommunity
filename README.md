# The Moroccan Community

Cette plateforme est dédiée à l'éducation financière et à l'investissement au Maroc.

## Fonctionnalités
- Analyses de marché par IA
- Boutique de produits numériques (E-books, templates)
- Outils financiers (Simulateurs, calculateurs)
- Espace membre et formations

## Installation et Développement

1. Installer les dépendances :
```bash
npm install
```

2. Lancer le serveur de développement :
```bash
npm run dev
```

## Sécurité
Ce projet utilise les standards de sécurité modernes. Ne jamais inclure de clés d'API ou de jetons d'accès dans le code source. Utilisez les variables d'environnement (`.env`) ou configurez vos accès Git via le terminal uniquement.
